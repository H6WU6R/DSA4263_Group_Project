# Project Documentation

Welcome to the DSA4263 Group Project documentation site.

## Getting Started

- Use `make install` to set up the Python environment.
- Populate `data/raw` with source datasets.
- Launch notebooks from the `notebooks/` directory for exploratory analysis.

## Contents

- **Data**: Information about sources, cleaning, and feature generation.
- **Modeling**: Baseline models, experiments, and evaluation summaries.
- **Reports**: Final deliverables and figures for presentations.
